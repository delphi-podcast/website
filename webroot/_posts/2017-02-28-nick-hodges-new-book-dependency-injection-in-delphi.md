If you've been following along, this is <a href="https://leanpub.com/dependencyinjectionindelphi">Nick's 3rd book</a> in recent years, following on the heals of <a href="http://codingindelphi.com/">Coding in Delphi</a> and <a href="http://morecodingindelphi.com/">More Coding in Delphi</a>. This time around he has teamed up with <a href="http://dsharp.org/">Embarcadero MVP Stefan Glienke</a> to cover the topic of Dependency Injection in depth and complete detail.
<blockquote><a href="https://leanpub.com/dependencyinjectionindelphi">This book</a> covers Dependency Injection from A to Z. In it you'll learn about Constructor Injection, Property Injection, and Method Injection. You'll learn about the right and wrong way to use the Dependency Injection Container.</blockquote>

<ul>
 	<li>What is Dependency Injection</li>
 	<li>Benefits of Dependency Injection</li>
 	<li>Constructor Injection</li>
 	<li>Property Injection</li>
 	<li>Method Injection</li>
 	<li>Using the Container</li>
 	<li>A Simple Migration Example</li>

 	<li>Dependency Injection Anti-patterns</li>
 	<li>A Full, Useful Example</li>
 	<li>Final Thoughts</li>
</ul>
