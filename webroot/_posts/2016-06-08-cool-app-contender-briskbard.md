<a href="http://community.embarcadero.com/blogs/entry/join-our-club-of-elite-embarcadero-developers-show-us-your-cool-apps"><img class="alignright wp-image-2449" src="http://delphi.org/wp-content/uploads/2016/05/Cool_App_Contest_Logo_and_Graphic.png" alt="Embarcadero Cool App Contest" width="206" height="198" /></a>I've been a convert to Chrome for a while now, but <a href="https://www.briskbard.com/">BriskBard</a> may just be the browser to convert me. It is written in Delphi, and is a contender for the <a href="http://community.embarcadero.com/blogs/entry/join-our-club-of-elite-embarcadero-developers-show-us-your-cool-apps">Embarcadero Cool App Contest</a>. BriskBard is a complete Internet suite capable of doing the most common tasks in Internet with only one program.

<a href="https://www.briskbard.com/"><img class="alignleft wp-image-2503 size-full" src="http://delphi.org/wp-content/uploads/2016/06/BriskBard.png" alt="BriskBard" width="200" height="200" /></a>
<blockquote><strong>BriskBard</strong> is the new web browser for Windows that includes an email client, a media player, a news aggregator, a contact manager, an FTP client, a newsgroups reader, instant messaging, and several web developer tools. BriskBard has all the features most of the people use and some advanced features for web developers. Its user interface is easy, fast and intuitive with many configuration options.</blockquote>
<a href="https://www.embarcadero.com/products/delphi"><img class="alignright" src="http://delphi.org/mvp/logo/PoweredBy/Delphi/Powered_by_Delphi.png" alt="Powered by Embarcadero Delphi" width="256" height="104" /></a>BriskBard is built with Embarcadero Delphi for Windows using <a href="http://docwiki.embarcadero.com/RADStudio/en/Indy">Indy</a>, <a href="http://docwiki.embarcadero.com/RADStudio/en/VCL_Overview">VCL</a>, <a href="http://docwiki.embarcadero.com/RADStudio/en/OpenSSL">OpenSSL</a>, <a href="http://docwiki.embarcadero.com/RADStudio/en/Sqlite">SQLite</a> and <a href="https://delphihaven.wordpress.com/hunspell/">Hunspell</a>.

It has an impressive <a href="https://www.briskbard.com/index.php?pageid=features">list of features</a>. Check out the <a href="https://www.youtube.com/watch?v=KPwvaMlJE3A">video</a> or download the fully featured 30 day trial in English and Spanish. Other languages can be added via the language manager.
