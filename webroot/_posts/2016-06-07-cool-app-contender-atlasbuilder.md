<a href="https://www.facebook.com/AtlasBuilder-946416705394449"><img class="alignnone wp-image-2488" src="http://delphi.org/wp-content/uploads/2016/06/AtlasBuilder-Wide.png" alt="Atlas Builder" width="354" height="131" /></a>
AtlasBuilder makes it easy to create an atlas and pack it into an encrypted db. Then generate a 'viewer' app. <img class="alignright" src="http://delphi.org/mvp/logo/PoweredBy/Delphi/Built_with_Delphi.png" alt="Built with Delphi" width="197" height="80" />

Built with Delphi for Windows using VCL, the Parallel Programming Library, FireDAC, DevExpress and TatukGIS DK.

https://youtu.be/SMmOyPVxsbM
