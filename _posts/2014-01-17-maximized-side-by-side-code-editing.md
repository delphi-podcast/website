Sometimes we forget some of the basics. Had a customer ask if you can maximize the code editor Window in RAD Studio and edit two files side-by-side. I'd heard someone talk about this a while ago, but I couldn't remember the details.

There is an option in Tools / Options / Editor Options / Display that allows a code window to <a href="http://docwiki.embarcadero.com/RADStudio/XE5/en/Display"><strong>Zoom to full screen</strong></a>.

<a href="http://docwiki.embarcadero.com/RADStudio/XE5/en/Display"><img class="aligncenter size-full wp-image-1474" alt="Zoom To Full Screen" src="http://delphi.org/wp-content/uploads/2014/01/ZoomToFullscreen.png" width="716" height="276" /></a>

Then right click in the code window you want full screen, bringing up the <a href="http://docwiki.embarcadero.com/RADStudio/XE5/en/View_Menu">View Menu</a>,Â and choose <a href="http://docwiki.embarcadero.com/RADStudio/XE5/en/New_Edit_Window">New Window</a><a href="http://docwiki.embarcadero.com/RADStudio/XE5/en/New_Edit_Window"><img class="aligncenter size-full wp-image-1475" alt="New Edit Window" src="http://delphi.org/wp-content/uploads/2014/01/NewEditWindow.png" width="538" height="365" /></a>

&nbsp;

With Windows 7 or Windows 8, just drag this new code window to the left or right edge of the screen and it automatically gets tiled nicely.
