XE7 is full of <a href="http://docwiki.embarcadero.com/RADStudio/XE7/en/What's_New_in_Delphi_and_C%2B%2BBuilder_XE7">new features</a> everywhere, but there are some really nice ones specific to Android. Here is a replay of my <a title="Skill Sprints" href="http://www.embarcadero.com/de/landing-pages/skill-sprints">Skill Sprint</a> session on New Android Features.

https://www.youtube.com/watch?v=qnMRXKvsDyg

The new Android specific features in XE7 include:
<ul>
	<li><a title="Adding Advertising to Your Mobile Apps" href="http://docwiki.embarcadero.com/RADStudio/en/Adding_Advertising_to_Your_Mobile_Apps">Google Mobile Ads SDK</a> (replacing AdMob for XE6 too)</li>
	<li><a href="http://docwiki.embarcadero.com/RADStudio/XE7/en/What's_New_in_Delphi_and_C%2B%2BBuilder_XE7#Full-Screen_Immersive_Mode_for_Android">Immersive Fullscreen mode</a> (for Android KitKat 4.4+)</li>
	<li><a href="http://docwiki.embarcadero.com/RADStudio/en/Application_Options#Splash_Images">Splash screen</a> support for Android</li>
	<li><a href="http://docwiki.embarcadero.com/RADStudio/en/Mobile_Tutorial:_Using_Remote_Notifications_(iOS_and_Android)">Push notifications</a> and advertising <a title="Entitlement List" href="http://docwiki.embarcadero.com/RADStudio/XE7/en/Entitlement_List">project options</a></li>
	<li>Project Manager support to <a href="http://docwiki.embarcadero.com/RADStudio/XE7/en/Adding_A_Java_Library_to_Your_Application_Using_the_Project_Manager">add Java libraries to app</a></li>
	<li><a href="http://docwiki.embarcadero.com/RADStudio/XE7/en/Java2OP.exe,_the_Native_Bridge_File_Generator_for_Android">Java2OP tool</a>: Generate Object Pascal interfaces for Java Android Classes (<a href="http://cc.embarcadero.com/item/30007">separate download</a>)</li>
</ul>
There are lots of other new features that are not specific to Android, but that will still help make your apps amazing for Android, iOS, Windows and OS X.
